# 助盲跑

连接盲人跑者与志愿者/跑团的平台。

## 快速预览

直接在浏览器中打开 `prototype.html` 即可查看移动端原型。

## 产品概述

盲人有跑步锻炼的需求，但缺乏安全跑步的陪伴支持。目前盲人跑步主要依赖跑团在固定地点的定期活动。

**目标**：建立一个连接盲人跑者与志愿者/跑团的平台，让盲人跑步需求的响应更高效、更规模化。

## 用户角色

| 角色 | 描述 |
|------|------|
| 盲人用户 | 浏览活动，报名参加，找到志愿者陪伴跑步 |
| 志愿者 | 加入跑团，发布活动，陪伴盲人跑步 |
| 跑团管理员 | 管理跑团成员，发布例跑活动，审核志愿者 |

## 一期功能

- [ ] 志愿者注册/登录 → 发布跑步活动
- [ ] 盲人浏览活动列表 → 报名参加
- [ ] 跑团入驻 → 审核志愿者 → 发布例跑活动
- [ ] 活动报名状态管理
- [ ] 活动完成后的双方评价
